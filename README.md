# Logiciel Club de Foot - Projet Vercel

## Installation
```bash
npm install
npm run dev
```

## Mise en ligne sur Vercel
1. Créez un dépôt GitHub avec ces fichiers.
2. Connectez ce dépôt à Vercel.
3. Vercel détectera automatiquement Vite.
4. Build command : `npm run build`
5. Output directory : `dist`

## Données
Les données sont stockées localement dans le navigateur via `localStorage`.
Chaque appareil garde ses propres données.
